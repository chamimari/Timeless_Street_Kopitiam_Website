# Timeless Street Kopitiam Website
A Web Application group project where students create a website with HTML, CSS and PHP, to provide full functionality. 

Timeless Street Kopitiam Website was created to help merge traditional and modern technology together, trying to keep traditional food culture such as kopitiam main stream and future-proof with a website, allowing online viewing, ordering and much more.

Project is created by me and one other member, whose name I shall not disclose for privacy reasons.

Disclaimer: The content, including all illustrations, renderings, and images presented in this project, is intended for non-profit, educational, and informational purposes only.
